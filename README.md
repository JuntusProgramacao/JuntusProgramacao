# Bem-vindo ao Repositório do Livro "Juntus Programação"

Este repositório é dedicado à armazenagem dos códigos apresentados no livro "Juntus Programação". O livro foi criado com o objetivo de fornecer um guia de algoritmos em várias linguagens de programação. Os exercícios abordados são de nível básico, projetados para serem compreensíveis por iniciantes na área de programação.

## Sobre o Livro

O "Juntus Programação" é uma iniciativa liderada pela professora Luciene Cavalcanti Rodrigues, em colaboração com diversos alunos. A intenção por trás do livro é proporcionar uma introdução prática e acessível aos fundamentos da programação, por meio de uma variedade de exemplos de exercícios.

## Contribuições

Este projeto foi possível graças à colaboração de alunos e entusiastas da programação. Se você deseja contribuir com exemplos de código, correções ou melhorias para o livro, sinta-se à vontade para abrir um pull request. Todas as contribuições são bem-vindas e ajudam a melhorar a qualidade do material disponível.
